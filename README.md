# Xplode

[![gitter](https://badges.gitter.im/gitterHQ/gitterHQ.github.io.svg)](https://gitter.im/Bytes_Club/General)

> A 2D Game based on OpenGL

### Prerequisutes

Install GLUT:

```
sudo apt-get install freeglut3-dev
```

### Compile and Run

Create bin/ directory:

```
mkdir bin/
make

cd bin/
./xplode
```
